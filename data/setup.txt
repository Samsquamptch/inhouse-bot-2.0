Welcome to the inhouse bot 2.0 setup! By using this command, you'll be able to set the channels the bot uses for your server.
Please press the "Configure Settings" button to add the ID for each required channel in the appropriate field. Once this is complete, press the "Confirm button" to finish setup.
The required channels are:
- Admin Channel
- Queue Channel
- Global Queue Channel
- Chat Channel
More details can be found on the GitHub setup page
